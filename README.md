# Maryam Yasser - Integrating With HubSpot I Practicum

This repo contains the Integrating With HubSpot I: Foundations Practicum project.

## HubSpot Custom Object list view
https://app-eu1.hubspot.com/contacts/147240851/objects/2-194504732/views/all/list

## How to run locally
1. Clone the repo
2. Run `npm install`
3. Create `.env` with:
   - `HUBSPOT_API_KEY=<your token>`
   - `CUSTOM_OBJECT_NAME=<your API name>`
4. Run `npm start`
